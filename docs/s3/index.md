# s3 or Simple Storage Service

s3 or AWS Simple Storage Service, hence the s3 since "s" is repeated 3 times, is a storage solution utilized in AWS account for storing objects, e.g. pdf, csv, tsv, xlsx, etc.
AWS s3 is a cheap an very reliable storage solution that has 11 9's of reliability or 99.99999999999.
